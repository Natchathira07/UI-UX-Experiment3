## 2.Heuristic Evaluation of HDFC and ICICI Banking Apps

1.This study evaluates the usability of HDFC Bank MobileBanking and ICICI iMobile using standard heuristic principles.

2.Both apps provide secure login, transaction tracking, and clear system feedback to users.

3.HDFC MobileBanking offers simpler navigation, making basic banking tasks easier to perform.

4.ICICI iMobile includes more advanced features, but the interface may appear slightly complex for new users.

5.Overall, both apps follow usability guidelines, but improvements in layout simplicity and feature organization can enhance user experience.

## 3.Testing the Core Features of Both Applications

1. Log in to both applications using secure authentication methods (PIN/biometric).

2. Check the dashboard to view account balance and recent transactions.

3. Test fund transfer and UPI payment features for ease of use.

4. Explore bill payment and recharge options.

5. Review profile settings, security options, and transaction history for usability.
